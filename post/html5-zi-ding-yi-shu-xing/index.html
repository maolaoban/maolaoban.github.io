HTML5自定义属性&&Vue事件委托

2021-04-08

1. HTML5自定义属性
在HTML5中提供了自定义属性，使用"data-"开头写在标签里，例如：

    <div data-name="maolaoban"></div>

在js中定义属性或者读取属性

    var div = document.querySelector("div");
    //自定义
    div.dataset.age = 20;
    //读取
    console.log(div.dataset.name);//div.dataset['name']
    console.log(div.dataset.age);

自定义属性规范：
1.data-开头；
2.data-后必须至少有一个字符，多个单词使用-连接；
建议：
1.名称应该都使用小写；
2.名称中不要包含任何的特殊符号；
3.名称不要纯数值

2. Vue事件委托

    <body>
    <div id="app">
        <ul @click="popName">
            <li v-for="(item,index) in list" :data-index="index">{{item.name}}:{{item.age}}</li>
        </ul>
    </div>
    <script src="./vue.js"></script>
    <script>
        var app = new Vue({
            el:'#app',
            data:{
                list:[
                    {
                        name:'mao',
                        age:20
                    },
                    {
                        name:'lao',
                        age:21
                    }
                    ,{
                        name:'ban',
                        age:22
                    }
                    ,{
                        name:'ha',
                        age:23
                    }
                ]
            },
            methods: {
                popName(event){
                    var e = event || window.event;
                    var target = e.target || e.srcElement;
                    if(target.nodeName.toLowerCase() == 'li'){
                        console.log(this.list[e.target.dataset.index].name);
                    }
                }
            },
        })
    </script>
    </body>
