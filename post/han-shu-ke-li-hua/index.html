函数柯里化

2021-04-08

首先，我们要知道什么叫柯里化？

在计算机科学中，柯里化（Currying）是把接受多个参数的函数变换成接受一个单一参数(最初函数的第一个参数)的函数，并且返回接受余下的参数且返回结果的新函数的技术。

  • 举一个简单的例子

    //普通函数
    function add(a + b){
    return a + b;
    }
    add(1,3);

    //函数柯里化
    function add(a){
    return function(b){
    return a + b;
    }
    }
    //我们可以这样使用
    add(1)(3)

如果要使用add(1)(2)(3)...(n),上面的写法也就无力回天了，这是我们就需要使用一个递归柯里化。

  • 简单的实现

    const curry = (fn, ...args) =>
    args.length < fn.length
    ? (...arguments) => curry(fn, ...args, ...arguments)
    : fn(...args);

     function sum (a,b,c){
     return a + b + c;
 }

 var add = curry(sum);

 console.log(add(1)(2)(3));//6
 console.log(add(1)(2,3));//6
 console.log(add(1,2)(3));//6
 console.log(add(1,2,3));//6
